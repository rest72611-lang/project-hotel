# Project Hotel

Vacation management system built with `React`, `TypeScript`, `Node.js`, `Express`, `MySQL`, `Docker`, AI recommendation tools, and MCP integration.

GitHub repository:

- `https://github.com/rest72611-lang/project-hotel`

Project structure:

- `Database` - database export and seed data
- `Backend` - REST API in Node.js + Express + TypeScript
- `Frontend` - React + TypeScript client
- `Postman` - exported Postman environment files and request definitions

Run with Docker:

```powershell
docker compose up -d --build
```
